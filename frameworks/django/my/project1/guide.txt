Projeto 1
Este projeto tem como objetivo desenvolver uma aplicação web que faça:
- Adição
- Edição
- Visualização
- Deleção

Banco de dados: SQLite
Tabelas:
- Produto (nome, descrição, preço, imagem)
- Fornecedor (CNPJ, nome, email, telefone)
- Cliente (CNPJ, nome, email, telefone)

Outras funcionalidades:
- Os produtos terão imagem

Outros:
- Por enquanto, não será publicado
- Superuser: admin 1234

Problemas:
- Ao cadastrar um produto cria várias vezes se o botão for clicado sucessivamente

Futuros desenvolvimento
- Cadastrar um código único do produto definid pelo usuário
- Melhorar o layout
